\# Finance Dashboard



Ứng dụng quản lý tài chính cá nhân xây dựng bằng Next.js + Supabase.



\## Tính năng



\- Dashboard

\- Transactions

\- Wallets

\- Budgets

\- Goals

\- Debts

\- Investments

\- AI Advisor

\- Reports

\- PWA



\## Công nghệ



\- Next.js 15

\- TypeScript

\- TailwindCSS

\- Supabase



\## Chạy local



```bash

cd frontend

npm install

npm run dev

```
